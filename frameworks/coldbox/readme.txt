Coldbox / Hyrule Sample Application
--------------------------------------------------------------------------------------------------------------------

	This is a sample application that shows off the integration of Hyrule and Coldbox. 

Requirements
--------------------------------------------------------------------------------------------------------------------

	ColdBox 3.1
	ColdFusion 9

Installation	
--------------------------------------------------------------------------------------------------------------------
	
	Mappings
	/coldbox 	(points to ColdBox 3.1 folder)
	/hyrule 	(you should already have this one in place)
